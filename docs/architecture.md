# Project description (excerpt)

**GASQ Laravel** is a web application for running a security-services marketplace: organizations can post work, vendors can participate, and both sides can use built-in pricing and operations calculators. The product combines marketing pages, user accounts, job listings, optional credit-based access to tools, and administrative features—implemented as a standard Laravel site with server-rendered pages and a conventional database-backed architecture.

For a **non-technical** audience, see **[OVERVIEW_FOR_STAKEHOLDERS.md](OVERVIEW_FOR_STAKEHOLDERS.md)**.

Technical documentation (setup, platform operation, Stripe, theme, spreadsheet mapping) is consolidated in **[README.md](README.md)** in this folder.
